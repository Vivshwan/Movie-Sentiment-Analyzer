# 🎬 IMDB Movie Review Sentiment Analyzer

A machine learning model that classifies IMDB movie reviews as **POSITIVE** or **NEGATIVE** with **~89% accuracy** using Logistic Regression and TF-IDF vectorization.

## 📊 Model Performance
- **Accuracy:** ~89%
- **Algorithm:** Logistic Regression
- **Features:** TF-IDF (5000 features)
- **Dataset:** 50,000 IMDB movie reviews

## 🚀 Features
- ✅ Clean and preprocess text (lemmatization, stopword removal)
- ✅ Train Logistic Regression model on 50,000 reviews
- ✅ Real-time sentiment prediction for any review
- ✅ High accuracy with efficient processing
